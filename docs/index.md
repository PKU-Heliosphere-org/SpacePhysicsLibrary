# 空间物理知识库

欢迎来到 `SpacePhysicsLibrary`。

这个知识库基于 MkDocs Material 搭建，侧边栏层级完全由根目录下的 `mkdocs.yml` 控制。新增知识点时，只需：

1. 在 `docs/` 对应分类目录中新建 Markdown 文件。
2. 在 `mkdocs.yml` 的 `nav` 中追加对应条目。
3. 刷新或重新启动 MkDocs 服务，侧边栏将自动更新。
